# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-07

### Added

- Initial GCP foundation platform repository.
- Python CLI for validation, tfvars generation, inventory rendering, repo checks, and Terraform command orchestration.
- Terraform root composition for folders, projects, Shared VPC, IAM, org policies, logging, and monitoring.
- Unit tests for the Python configuration and automation layer.
- GitHub Actions CI and Terraform plan workflows.
- Architecture docs, ADRs, runbooks, and example organization configuration.
